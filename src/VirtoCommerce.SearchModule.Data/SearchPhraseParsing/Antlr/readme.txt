# How to build Antrl grammar file for search phrase expression

SearchPhrase.g4 file contains the grammar for the search phrase expression from keyword.

## How to generate a parser for Windows:
1. It is assumed that you have installed Java.
2. Run generate.bat

## Update Path:
1. Go to http://www.antlr.org and download the latest version of antlr-4.13.2-complete.jar.
2. Update Antlr4.Runtime.Standard to latest version.
